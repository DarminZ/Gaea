#### Gaea: 希腊神话中的人物，大地女神，紧随夜之女神出现的神，世界的缔造者之一，创造了大地，海洋和天空。

####目录结构
```
├── build                                       // webpack配置文件
├── config                                      // 项目打包路径
├── src                                         // 源码目录
│   ├── assets                                  // 图片资源
│   ├── components                              // 组件
│   ├── config                                  // 基本配置
│   ├── pages                                   // 页面
│   ├── router                                  // 路由
│   └── style                                   // 样式
│   ├── App.vue                                 // 页面入口文件
│   ├── main.js                                 // 程序入口文件，加载各种公共组件
├── index.html                                  // 入口html文件
```
## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
